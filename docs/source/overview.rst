Requirements
============
.. code-block:: bash
 #. PHP >= 7.2

.. _installation:


Installation
============
.. code-block:: bash

    composer require crazymeeks/oauth0 v1.0.0

Alternatively, you can specify oauth0 as a dependency in your project's
existing composer.json file:

.. code-block:: js

    {
      "require": {
         "crazymeeks/oauth0": "^v1.0.0"
      }
   }
